Introduction

  • This project is a simple RESTful API built using NodeJS, KOA, Typescript and TypeORM as an ORM (with MongoDB as a database). Middleware uses JWT, CORS, in-memory rate limiter, Winston Logger.
  • Swagger API docs are used to produce the API front-end you see right now

How to use

  • This front-end is live and the API is hosted on Heroku. Mongo Atlas free tier is used as the mongoDB database.
  • Create a new user using POST /user , you can use the sample data.
  • With the credentials you just used (email and password), log the user in via POST /login
  • In the response you will receive a payload with a token. Copy this token.
  • At the top of the page, click Authorize and enter Bearer the_token_you_copied in the value box and click authorize e.g. Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9
  • Now you are authorized to use access the rest of the routes
  • After about 15 min th token will expire and you can use GET /refresh to get a new token. The refresh token lasts for 24 hours. After that you will have to log in again.
  • You can play around with the rest of the routes. You can also delete all created users with DELETE /testusers and start afresh.
